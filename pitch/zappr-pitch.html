06 / 10
how it works
Architecture

Sovereign by default. Friendly by design.

Nostr · NDK

Identity, follows, relays. We never see your nsec.

Breez SDK

Lightning + Liquid in the browser via WASM. Seedless wallets.

NWC

Power users bring Alby, Mutiny, Primal. One string, plug in.

Claude Agent

Reads, drafts, explains. Multilingual. Confirms payments.

One passkey. Two keys. Zero custody.

PRF(passkey, "NYOASTRTSAOYN") → BIP32 m/44'/1237'/55'/0/0 → Nostr nsec
PRF(passkey, "zappr-liquid-v1") → BIP39 mnemonic → Breez Lightning wallet

Same fingerprint, both keys reproduced on every unlock. Nothing stored except the credential id. Logout clears it. Backup phrase exportable for power users who want to graduate.
